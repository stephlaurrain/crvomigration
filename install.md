# Commands

## Create virtual env

python3 -m venv env

## Activate virtual env 

source env/bin/activate

## Install dependencies

pip3 install -r requirements.txt

## Save dependencies

pip freeze > requirements.txt


## env version

pipenv --version

## Virtual env location

pipenv --venv

## Destroy it

pipenv --rm

## Save dependencies

pip freeze > requirements.txt

## UPDATE

pip install --upgrade -r requirements.txt
